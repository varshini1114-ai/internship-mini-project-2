# Digital Diary with PIN Lock

## 📌 Project Overview

Digital Diary with PIN Lock is a secure personal journal application that allows users to write, edit, and manage diary entries. Access to the diary is protected using a PIN authentication system, ensuring privacy and security. All diary entries are stored in a JSON file for easy data management and persistence.

## 🚀 Features

* Secure PIN-based login system.
* Create, edit, and delete diary entries.
* Store diary entries in a JSON file.
* View previous journal entries.
* Simple and user-friendly interface.
* Data persistence without a database.

## 🛠️ Technologies Used

* HTML
* CSS
* JavaScript
* JSON File Storage

## ⚙️ How to Run the Project

1. Download or clone the repository.
2. Open the project folder.
3. Ensure the JSON file is present for storing entries.
4. Open `index.html` in your browser.
5. Enter the correct PIN to access the diary.
6. Create, edit, or delete diary entries as needed.
